Pictures from Spring in Kyoto, 3/26/26

Hi so the blog post today is that I'm trying a new way to post pictures
online for free, and display them how I like them. So I've uploaded some
pics to imgur, and have made a tiny stylesheet that displays them in a
grid. I don't really like the google photo albums for this purpose, as there is
a storage limit. I also don't really like the layout of imgur galleries, as I
like seeing all my pics in a grid (and optional captions etc.).
